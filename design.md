# Jack Programming Language

* Data flow programming
* Purely functional
* Strong, static, and inferenced type system
  * Primitives (Int, Real), structure, array, and time-oriented queue
* No concept of the time


## built-ins

* +, -, \*, /, \*\*, //
* &, |, ^ (bitwise)
* ==, /=, <, >, <=, >=
* map
